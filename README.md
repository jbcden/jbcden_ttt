# JbcdenTtt

## An unbeatable game of tic-tac-toe!

## Installation

- Simply run:
```bash
gem install jbcden_ttt
```

- And then play!:
```bash
ttt
```

## Usage

Please note that this gem does require Ruby 1.9 or later.

Also note that this gem is based on the repo:

https://github.com/jbcden/tic-tac-toe
